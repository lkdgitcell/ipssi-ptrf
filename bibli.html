Projet Bibliothèque (SLAM)

Site web de gestion de bibliothèque avec interface utilisateur moderne

Projet Étudiant 2024–2025

Vue d'ensemble

Ce projet consiste au développement d'un site web complet pour exposer une bibliothèque en ligne. L'objectif était de créer un système permettant la gestion des livres. Le projet met l'accent sur les bonnes pratiques de développement et les conventions d'architecture.

Objectifs

Maîtriser le développement web en tant que débutant, comprendre les concepts de base de données, et développer les compétences en programmation. Le projet vise également à créer une interface utilisateur responsive et accessible tout en respectant les standards web.

Réalisations

Interface Utilisateur

Création d'une interface moderne et responsive avec HTML/CSS et PHP

Base de Données

Conception et implémentation d'une base de données relationnelle via MySQL

Fonctionnalités

  • Gestion complète du catalogue de livres
  • Interface d'administration intuitive
  • Recherche dynamique et filtrage avancé

Résultats

10+ Livres gérés
